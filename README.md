# AgeMaster

**AgeMaster** is a C++ console application designed to calculate your age based on your birth date and the current date. It also includes functionality to calculate the age difference between two people. This project demonstrates the use of functions, loops, and basic input/output in C++.

## Features

- **Age Calculator:**  
  Calculate your current age in years, months, and days based on your birth date and a given current date.

- **Age Difference (Planned):**  
  Placeholder functionality to calculate the difference between two individuals’ ages.

## Prerequisites

- C++ compiler (e.g., GCC, MinGW, or Visual Studio)
- Basic knowledge of running console programs
